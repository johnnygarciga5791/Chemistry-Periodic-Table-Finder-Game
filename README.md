# Chemistry Periodic Table Finder & Quiz

An interactive web application that allows users to search for chemical elements, explore their properties, and test their knowledge through quizzes.

The project helps reinforce chemistry concepts such as element symbols, names, atomic numbers, and element families through interactive learning.

---

## Features

### Element Search
Users can search for an element by:
- Element **name**
- Element **symbol**
- **Atomic number**

The application displays:
- Element name and symbol
- Atomic number
- Atomic mass
- State of matter (solid, liquid, gas)
- Element family (alkali metal, noble gas, etc.)

---

### Interactive Quizzes

#### Type-In Quiz
Users manually type the correct answer.

Question types include:
- Element **symbol from name**
- Element **name from symbol**

#### Multiple Choice Quiz
Users select the correct answer from four randomized options.

Features:
- Randomized questions
- Randomized answer order
- Score tracking
- Instant feedback

---

### Periodic Table Viewer
Users can view all stored elements using the **Show All** button.

Elements are visually styled based on their state of matter:
- **Solid**
- **Liquid**
- **Gas**
